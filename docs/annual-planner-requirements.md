# 年度计划 — 功能需求

## 概述

一个用于制定、拆解和追踪年度目标的工具。用户按领域组织目标，将目标拆解为可勾选的检查项，通过标签关联个人能力维度，实时掌握全年进展。

## 核心概念

| 概念 | 说明 |
|------|------|
| **领域** | 目标的分组，如事业、财务、健康、兴趣。用户可增删改，支持排序 |
| **目标** | 归属于某个领域的具体年度目标，有标题、描述、优先级 |
| **检查项** | 目标下的具体行动步骤，可勾选完成，记录完成时间 |
| **标签** | 跨领域的能力维度标记，如"法语"、"管理"、"体能"，可关联到任意目标 |

## 功能清单

### 1. 领域管理

- 默认提供四个领域：事业、财务、健康、兴趣
- 用户可新增、重命名、删除领域
- 删除领域时需确认，该领域下所有目标一并删除
- 领域支持拖拽排序

### 2. 目标管理

- 在指定领域下创建目标，填写标题（必填）和描述（选填）
- 设置优先级：高 / 中 / 低
- 可关联一个或多个标签
- 可编辑、删除目标
- 删除目标时需确认，其下所有检查项一并删除
- 目标支持在同一领域内拖拽排序

### 3. 检查项管理

- 在目标下添加检查项，填写内容（必填）
- 勾选/取消勾选检查项
- 勾选时自动记录完成时间，取消勾选时清除完成时间
- 可编辑、删除检查项
- 检查项支持拖拽排序

### 4. 标签管理

- 创建标签（名称唯一）
- 编辑、删除标签
- 删除标签时仅解除关联，不影响目标本身
- 在目标上添加/移除标签

### 5. 总览页

- 展示所有领域，每个领域显示：
  - 领域名称
  - 目标数量
  - 整体完成率（已完成检查项 / 总检查项）
  - 进度条可视化
- 全局完成率：所有检查项的总完成比例
- **停滞提醒**：目标下所有检查项最近 14 天无任何勾选变动时，标记为"停滞"，视觉上高亮区分

### 6. 领域详情页

- 展示该领域下所有目标
- 每个目标显示：
  - 标题、描述、优先级标记
  - 关联的标签
  - 检查项列表及各项完成状态
  - 完成进度（如 3/7）
  - 停滞标记（如适用）

### 7. 标签聚合视图

- 按标签分组展示关联的所有目标（跨领域）
- 每个标签显示关联目标数量和整体完成率
- 用于回答"我在某项能力上投入了多少、进展如何"

## 交互规则

- 所有删除操作需二次确认
- 检查项勾选为乐观更新（先更新 UI，后同步服务端）
- 空状态需有引导提示（如"还没有目标，点击添加"）
- 进度为 100% 的目标视觉上标记为已完成

## 不做（V1 范围外）

- 跨年对比 / 历史归档
- 能力雷达图
- 截止日期 / 日历集成
- 多人协作
- 通知推送
