# 能力画像系统 — 使用手册

## 是什么

一个以可量化技能为基础的自我成长系统。通过里程碑解锁和能力验证追踪真实进步，像游戏角色面板一样了解自己的能力分布。

核心逻辑：**添加技能 → 完成里程碑 → 段位解锁 → 雷达图自动更新**

---

## 快速开始

### 1. AI 快速建档（推荐）

首次进入，点击 **AI 快速建档**。教练会通过对话了解你已有的技能和水平，自动创建技能档案并设置初始段位。整个过程约 5 分钟。

### 2. 手动添加技能

点击 **添加技能**，可以：
- **从模板创建** — 选择官方模板（法语、跑步、编程等），自动生成完整的里程碑体系
- **自定义创建** — 选择所属大类，输入技能名称

---

## 核心功能

### 雷达图

Dashboard 顶部的 7 维雷达图展示你在各能力大类的综合水平：

| 大类 | 说明 |
|------|------|
| 语言能力 | 外语听说读写 |
| 数理逻辑 | 编程、数学、数据分析 |
| 身体运动 | 跑步、游泳、力量训练 |
| 艺术创作 | 音乐、绘画、摄影 |
| 专业技能 | 职业相关的专业能力 |
| 生活实践 | 烹饪、驾驶等实用技能 |
| 自我管理 | 习惯坚持度、目标执行力（系统自动追踪） |

得分 = 该大类下所有技能的段位平均值。虚线表示数据不足（技能太少）。

### 技能与段位

每个技能有 5 个段位：

| 段位 | 星级 | 含义 |
|------|------|------|
| 入门 | ★ | 初学 1-3 个月 |
| 基础 | ★★ | 坚持半年到一年 |
| 胜任 | ★★★ | 认真训练 2-3 年 |
| 精通 | ★★★★ | 专业级或多年深入实践 |
| 卓越 | ★★★★★ | 顶尖或公认成就 |

**升级规则**：完成当前段位的所有里程碑 → 自动解锁下一段位。

### 里程碑

里程碑是段位升级的具体条件。每个里程碑有验证方式：

| 验证方式 | 说明 | 示例 |
|----------|------|------|
| 平台自动 | 系统自动查询数据判定 | 词汇量达到 500 |
| 平台验证 | 系统生成测试题 | 词汇抽查正确率 > 80% |
| 证据提交 | 上传截图或证书 | DELF B1 成绩单 |
| 自评声明 | 用户自行确认 | 能独立完成日常对话 |

**AI 生成里程碑**：在技能详情页，当里程碑少于 5 个时，可点击"AI 生成里程碑"自动生成。

**平台验证**：对于"平台自动"类里程碑，可点击验证按钮让系统自动检查（如查询实际词汇掌握数量、习惯打卡天数等）。

### 当前状态

除了历史成就（段位），每个技能还追踪"当前状态"——你现在的水平。

- 当前状态会**过期**（默认 180 天），过期后雷达图得分会打折
- 状态旁会显示距过期的天数
- 需要定期确认或更新

### 焦点计划

同时追踪 1-3 个最想提升的技能：

1. 在 Dashboard 的焦点计划区域点击创建
2. 选择技能、目标段位、截止日期
3. 可点击"生成策略"让 AI 分析并给出提升建议
4. 超过 7 天没有活动会收到停滞提醒

### 徽章

完成特定成就自动获得徽章。在 **荣誉墙** 查看所有徽章及达成条件。

徽章稀有度：普通 → 稀有 → 史诗 → 传说

## AI 教练

点击 **AI 教练** 进入对话。教练的风格：严格、诚实、以证据说话，不恭维。

教练会：
- 基于你的技能数据和焦点计划给出建议
- 记住之前对话的要点
- 在你"刷数据"时直接指出

快捷键：`Ctrl + Enter` 发送消息。

---

## 通知系统

Dashboard 顶部的横幅和底部的通知列表展示待处理事项：

- **停滞提醒** — 焦点技能超过 7 天无活动
- **状态过期** — 当前状态即将过期，需要确认
- **周回顾** — 每周一自动生成，汇总本周活动
- **月报** — 每月 1 日生成，含雷达快照对比

通知有预算限制（最多 1 条高优先 + 2 条中低优先），不会轰炸你。

侧边栏的能力画像图标旁会显示未读通知数量。

---

## 成长记录

点击 **成长记录** 查看：

- **成长曲线** — 各大类雷达得分的月度变化趋势
- **活动时间线** — 所有活动日志（里程碑完成、技能添加、习惯打卡等）

---

## 自我管理（自动）

"自我管理"大类下的技能由系统自动创建和追踪，无需手动操作：

| 技能 | 数据来源 |
|------|----------|
| 习惯坚持度 | 习惯打卡模块的连续天数 |
| 目标执行力 | 年度计划的目标和完成率 |
| 学习持续性 | 技能学习模块的活跃度 |

这些技能的里程碑也可以通过"平台验证"自动判定。

---

## 与其他模块的关系

能力画像不替代现有模块，而是在它们之上增加方向和动力：

```
能力画像（我要提升什么？下一个里程碑是什么？）
    ├── 习惯打卡 → 自动追踪连续性，反映到自我管理
    ├── 年度计划 → 自动追踪完成率，反映到自我管理
    ├── 技能学习 → 自动追踪知识点掌握度
    ├── 词汇学习 → 自动追踪词汇量，可自动验证里程碑
    └── 文章阅读 → 阅读量记录
```

你在各模块的活动会自动记录到能力画像的活动日志中。
