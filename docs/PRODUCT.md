## 产品概览

### 定位

面向个人用户的一站式成长管理平台，覆盖「目标规划 → 习惯养成 → 技能学习 → 知识积累 → 项目落地」的完整成长链路。

### 核心理念

- **数据驱动**：所有模块的进度、打卡、学习记录汇聚为统一数据，AI 教练可跨模块洞察
- **极简设计**：黑白单色视觉系统，CSS 变量控制主题，无视觉干扰
- **可扩展**：工具插件系统 + 技能学习引擎，新模块/新技能可低成本接入
- **本地优先**：SQLite 数据库本地部署，PM2 进程管理，隐私可控

### 技术栈

| 层级 | 技术 |
|------|------|
| 前端框架 | Nuxt 3 (SPA 模式) + Vue 3 Composition API |
| 状态管理 | Pinia (Composition API 风格) |
| 数据库 | SQLite (better-sqlite3) + Drizzle ORM |
| 认证 | JWT (365 天有效期) |
| AI | 插件式 LLM 接入 (Claude CLI / Ollama) |
| 部署 | PM2 进程管理 + 自动备份 |

## 系统架构概要

```
┌─────────────────────────────────────────────────┐
│                    前端 (SPA)                     │
│  ┌──────────┐  ┌──────────┐  ┌──────────┐       │
│  │  Pages    │  │  Stores  │  │Composable│       │
│  │ [...slug] │  │  (Pinia) │  │  (Auth,  │       │
│  │  → Tool   │  │  8 个    │  │  LLM..)  │       │
│  └──────────┘  └──────────┘  └──────────┘       │
│  ┌──────────────────────────────────────┐       │
│  │     Tool Plugin System (8 工具)       │       │
│  │  registerTool() → Map → [...slug]    │       │
│  └──────────────────────────────────────┘       │
│  ┌──────────────────────────────────────┐       │
│  │     小爽助手 (全局浮层)                │       │
│  └──────────────────────────────────────┘       │
├─────────────────────────────────────────────────┤
│                 Nitro Server                     │
│  Middleware: CacheControl → Log → Auth → Guard  │
│  150+ API Routes (file-based routing)           │
│  LLM Providers (Claude CLI / Ollama)            │
│  Skill Learning Core (通用学习引擎)              │
├─────────────────────────────────────────────────┤
│              SQLite (WAL mode)                   │
│  13 Schema 文件 · Drizzle ORM · 自动迁移        │
└─────────────────────────────────────────────────┘
```

### 工具插件系统

每个工具模块是一个独立目录 (`tools/<id>/`)，包含：
- `index.ts` — 调用 `registerTool()` 注册工具元信息 (名称、图标、排序)
- 根组件 `.vue` — 延迟加载的工具界面
- `components/` — 工具内部子组件

路由自动解析：URL `/habit-tracker` → 查找 ID 为 `habit-tracker` 的工具 → 渲染对应组件。

## 模块详解

### 1. 今日 (Dashboard)

**入口**：侧边栏首项，应用启动默认页面

**功能概述**：
聚合所有模块的当日关键数据，提供一目了然的每日概览和 AI 驱动的个性化洞察。

**核心功能**：

| 功能 | 说明 |
|------|------|
| 每日问候 | 根据时间段显示问候语，附带当日摘要统计 |
| AI 洞察 | 基于全模块数据生成个性化每日建议 |
| 今日议程 | 展示当日待办目标和计划项 |
| 快速统计 | 打卡连续天数、学习进度、目标完成率等关键指标 |
| 近期动态 | 跨模块的最近活动流 |

**使用方式**：
1. 打开应用自动进入今日页面
2. 查看顶部问候横幅了解当日概况
3. 阅读 AI 洞察获取个性化建议
4. 通过快速统计卡片快速跳转到对应模块

### 2. 能力画像 (Ability Profile)

**入口**：侧边栏「能力画像」

**功能概述**：
构建个人能力模型，通过 7 维雷达图可视化技能分布，使用里程碑分级体系追踪每项技能的成长路径。

**核心功能**：

| 功能 | 说明 |
|------|------|
| 7 维能力分类 | 技术能力、产品思维、商业洞察、领导力、沟通表达、创造力、学习能力 |
| 技能管理 | 添加技能（从模板或自定义），归类到能力维度 |
| 里程碑体系 | 每个技能最多 8 个等级（T1-T8），由低到高逐级解锁 |
| 里程碑验证 | 提交证据（文字描述/链接）完成里程碑，支持 AI 生成里程碑建议 |
| 雷达图 | 7 维 SVG 雷达图展示当前能力分布 |
| 成长曲线 | 月度快照记录能力变化趋势 |
| 成长时间线 | 按时间轴展示关键成长事件 |
| 专注计划 | 设定当前重点发展的技能，AI 生成发展策略 |
| 荣誉墙 | 完成里程碑获得的成就徽章 |
| AI 引导 | 初次使用时 AI 对话帮助发现和定义技能 |

**使用方式**：

1. **首次使用**：进入模块后，AI 引导对话帮助你梳理当前技能
2. **添加技能**：点击「+」从模板库选择或自定义技能，选择所属能力维度
3. **设定里程碑**：为每个技能定义分级里程碑（如 T1：入门、T2：熟练...），或使用 AI 自动生成
4. **完成里程碑**：提交完成证据，系统验证后解锁下一级（不可跳级）
5. **查看画像**：雷达图实时反映各维度能力水平
6. **设定专注**：选择 1-3 个技能作为当前发展重点，获取 AI 建议

**注意事项**：
- 里程碑必须按顺序完成，T3 需先完成 T1 和 T2
- 荣誉墙徽章与技能里程碑自动同步

### 3. 年度计划 (Annual Planner)

**入口**：侧边栏「年度计划」

**功能概述**：
按年度组织个人目标，将大目标拆解为可执行的检查项，支持跨年度复制和多维度统计。

**核心功能**：

| 功能 | 说明 |
|------|------|
| 多年管理 | 支持多年度规划，年份切换 |
| 领域分组 | 将目标按生活领域组织（如事业、健康、学习） |
| 目标管理 | 每个领域下创建具体目标，设定优先级 (高/中/低) |
| 检查项 | 目标拆解为可勾选的执行项 |
| 标签系统 | 跨目标标签分类，支持按标签视图聚合 |
| 年度复制 | 将上一年度结构复制到新年度 |
| 进度统计 | 总览/按领域/按标签的完成率统计 |
| 领域排序 | 拖拽调整领域和目标的展示顺序 |

**使用方式**：

1. **创建领域**：点击「+」添加生活领域（如「职业发展」「健康运动」）
2. **添加目标**：在领域下创建具体目标，设定优先级和标签
3. **拆解执行**：为每个目标添加检查项，逐一勾选完成
4. **查看进度**：切换到「总览」查看各领域完成率图表
5. **标签视图**：切换到「标签」页面，按标签维度查看关联目标
6. **跨年复制**：新年时一键复制上年结构，保留领域和目标框架

### 4. 日历打卡 (Habit Tracker)

**入口**：侧边栏「日历打卡」

**功能概述**：
创建习惯并在日历上打卡，追踪连续天数和完成率，通过热力图和趋势图可视化习惯养成过程。

**核心功能**：

| 功能 | 说明 |
|------|------|
| 习惯创建 | 定义习惯名称和频率 (每日/每周/每月) |
| 日历打卡 | 在月度日历上点击日期完成打卡 |
| 连续统计 | 当前连续天数和最长连续天数 |
| 月完成率 | 当月打卡完成率百分比 |
| 热力图 | 类 GitHub 贡献图的年度打卡热力图 |
| 趋势分析 | 历史打卡趋势折线图 |
| 乐观更新 | 打卡操作立即反映在 UI，后台异步同步 |

**使用方式**：

1. **创建习惯**：点击「+ 新习惯」输入名称和频率
2. **每日打卡**：在左侧习惯列表选择习惯，在右侧日历上点击今天的日期完成打卡
3. **查看统计**：日历上方显示当前连续天数和月完成率
4. **分析趋势**：点击「历史」查看长期趋势图和热力图
5. **管理习惯**：编辑或删除不再需要的习惯

### 5. 事项追踪 (Project Tracker)

**入口**：侧边栏「事项追踪」

**功能概述**：
轻量级项目管理工具，支持项目全生命周期管理，包括清单、笔记、图表、里程碑和 AI 辅助功能。

**核心功能**：

| 功能 | 说明 |
|------|------|
| 项目管理 | 创建项目，设定状态 (想法/待办/进行中/阻塞/完成/放弃) |
| 分类与标签 | 项目按分类组织，标签实现灵活筛选 |
| 清单系统 | 项目下创建待办清单，支持优先级和截止日期 |
| 里程碑 | 定义项目关键节点，追踪整体进度 |
| 笔记 | Markdown 笔记，支持附件上传和 AI 摘要 |
| 图表 | 生成项目相关图表（Mermaid 等格式） |
| 项目聊天 | 项目级 AI 对话，讨论方案和解决问题 |
| 进度概览 | 项目完成度统计 |

**使用方式**：

1. **创建项目**：点击「+」输入项目名称、描述，选择分类
2. **设定状态**：项目状态流转：想法 → 待办 → 进行中 → 完成（或阻塞/放弃）
3. **添加清单**：在项目详情页添加待办事项，设定优先级
4. **记录笔记**：使用笔记功能记录项目思考、会议纪要
5. **跟踪里程碑**：定义关键节点，检查整体进度
6. **AI 辅助**：通过项目聊天获取 AI 建议

### 6. 文章阅读 (Article Reader)

**入口**：侧边栏「文章阅读」

**功能概述**：
输入 URL 抓取文章内容，提供分屏阅读界面，支持 LLM 驱动的全文翻译、摘要生成和互动问答。

**核心功能**：

| 功能 | 说明 |
|------|------|
| 文章抓取 | 输入 URL 自动提取文章内容 |
| 手动录入 | 支持手动粘贴文章内容 |
| 分屏阅读 | 左侧原文 + 右侧翻译/笔记/聊天 |
| AI 翻译 | LLM 流式翻译（SSE 实时输出） |
| 文章摘要 | AI 生成文章核心内容摘要 |
| 书签系统 | 收藏文章并添加备注 |
| 文章聊天 | 基于文章内容的 AI 问答 |
| 标签管理 | 为文章添加分类标签 |
| 阅读历史 | 浏览历史记录 |

**使用方式**：

1. **导入文章**：在顶部输入框粘贴文章 URL，或手动录入
2. **阅读原文**：左侧面板显示提取的文章内容
3. **翻译文章**：右侧切换到「翻译」标签，点击翻译按钮启动 AI 流式翻译
4. **生成摘要**：选择「摘要」模式快速了解文章要点
5. **互动问答**：切换到「聊天」标签，就文章内容提问
6. **收藏管理**：点击书签图标收藏，添加备注方便日后查找

### 7. 法语词汇 (Vocab Tracker)

**入口**：侧边栏「法语词汇」

**功能概述**：
法语词汇学习工具，支持 CSV 批量导入，使用 SRS（间隔重复系统）科学记忆，LLM 生成词汇释义。

**核心功能**：

| 功能 | 说明 |
|------|------|
| CSV 导入 | 批量导入法语词汇列表 |
| 多用户 | 支持多个学习者独立进度 |
| 学习状态 | 未读 → 待学 → 学习中 → 已掌握 四阶段 |
| SRS 系统 | 间隔重复算法安排复习计划 |
| 每日计划 | 基于 SRS 算法推荐当日学习/复习词汇 |
| AI 释义 | LLM 生成词汇定义、例句（带缓存） |
| 分类视图 | 快速浏览和分类词汇 |
| 学习模式 | 沉浸式学习界面 |
| 进度图表 | 学习进度可视化 |
| 语音朗读 | Web Speech API 法语发音（优先法语语音） |

**使用方式**：

1. **导入词汇**：点击「导入」上传 CSV 文件（格式：法语词, 中文释义）
2. **分类词汇**：使用分类视图将词汇标记为「待学」
3. **每日学习**：查看 SRS 推荐的当日学习/复习列表
4. **学习模式**：进入沉浸式学习界面，查看释义、听发音
5. **查看释义**：点击词汇卡片查看 AI 生成的详细释义和例句
6. **追踪进度**：查看进度图表了解整体学习状况

### 8. 技能管理 (Skill Manager)

**入口**：侧边栏底部「技能管理」

**功能概述**：
动态创建和管理学习技能树。通过向导界面定义技能的知识结构、AI 教学提示词，创建后自动注册为独立的学习工具。

**核心功能**：

| 功能 | 说明 |
|------|------|
| 技能创建向导 | 分步定义技能名称、描述、知识树结构 |
| AI 知识树生成 | LLM 自动生成领域-主题-知识点的树状结构 |
| 提示词定制 | 自定义教学、聊天、练习、测验的 AI 提示词 |
| 技能列表 | 查看和管理所有已创建的学习技能 |
| 导入导出 | 将技能配置导出为 JSON，或从 JSON 导入 |
| 动态注册 | 创建的技能自动出现在侧边栏作为独立工具 |

**使用方式**：

1. **创建技能**：点击「新建技能」进入创建向导
2. **定义结构**：输入技能名称和描述，使用 AI 自动生成知识树，或手动编辑
3. **配置提示词**：根据需要调整 AI 教学风格和内容生成提示词
4. **确认创建**：完成后技能自动注册并出现在侧边栏
5. **开始学习**：点击侧边栏中新出现的技能工具，进入学习界面

### 9. 技能学习引擎 (Skill Learning Core)

**说明**：这不是一个独立的用户模块，而是支撑所有「技能学习」类工具的通用引擎。通过技能管理器创建的每个技能都使用此引擎。

**功能概述**：
从创业地图工具中抽取的通用结构化学习系统，所有技能工具共享同一套数据库表（按 skillId 隔离）、API 路由和 UI 组件。

**核心能力**：

| 能力 | 说明 |
|------|------|
| 知识树 | 三级结构：领域 → 主题 → 知识点 |
| AI 教学 | 结构化教学内容生成 (What/How/Example/Apply/Resources)，SSE 流式输出 |
| AI 聊天 | 知识点级别的问答对话 |
| 练习任务 | AI 生成的实践练习，完成状态追踪 |
| 理解测验 | AI 出题 + 自动评分 + 多次尝试记录 |
| 学习笔记 | 每个知识点的个人笔记 |
| 学习状态 | 未读/学习中/已掌握 三阶段状态管理 |
| 学习阶段 | 可定义学习阶段时间线 |
| 文章关联 | 将文章阅读器中的文章双向关联到知识点 |
| 活动记录 | 学习活动日志（小时级去重） |
| 热力图 | 学习活动热力图 + 连续天数 |
| 统计面板 | 整体/按领域的学习进度统计 |
| 推荐引擎 | 智能推荐下一个学习知识点 |

**22 个共享 UI 组件**：
包括 DomainCard、PointPage、TeachingContent、ChatPanel、PracticeTasks、ComprehensionQuiz、NoteEditor、LearningHeatmap、LinkedArticles 等，提供一致的学习体验。

### 10. 小爽助手 (Xiaoshuang AI Coach)

**入口**：页面右下角浮动按钮，任何页面可呼出

**功能概述**：
全局 AI 成长教练，能感知所有模块的数据上下文，提供跨模块的个性化成长建议和反馈。

**核心功能**：

| 功能 | 说明 |
|------|------|
| 全局可用 | 在任何工具页面都可以呼出对话 |
| 跨模块感知 | 自动收集习惯打卡、年度目标、技能进度、能力画像等数据作为上下文 |
| 多轮对话 | 支持连续对话，保持上下文 |
| 证据驱动 | 基于实际数据给出建议，而非泛泛而谈 |
| 未读提示 | 有新消息时显示未读计数 |

**感知的数据维度**：
- 习惯打卡记录与连续天数
- 年度计划进度与目标完成率
- 技能学习状态与活动频率
- 能力画像等级与里程碑进展
- 法语词汇学习进度
- 文章阅读历史
- 项目追踪状态

**使用方式**：

1. **呼出对话**：点击右下角的小爽图标
2. **提问或聊天**：输入你的问题、困惑或想法
3. **获取建议**：小爽会结合你的全模块数据给出针对性建议
4. **关闭对话**：点击关闭按钮或点击外部区域

**使用场景示例**：
- 「我最近学习状态怎么样？」→ 小爽分析打卡、学习活动、目标进度
- 「帮我制定下周计划」→ 结合年度目标和当前进度建议
- 「我的法语学习有什么建议？」→ 基于词汇进度和学习频率给出建议

## LLM 集成

### 支持的提供者

| 提供者 | 接入方式 | 状态 |
|--------|----------|------|
| Claude | 通过 `claude` CLI 调用 (需本地安装并认证) | ✅ 可用 |
| Ollama | REST API 调用 `http://localhost:11434` (需本地运行) | ✅ 可用 |
| OpenAI | Schema 和 UI 已存在 | ⏳ 未实现 |

### LLM 使用场景

| 场景 | 说明 |
|------|------|
| 文章翻译 | SSE 流式输出，实时翻译 |
| 文章摘要 | 提取文章核心内容 |
| 词汇释义 | 生成法语词汇详细释义（结果缓存） |
| AI 教学 | 技能学习的结构化教学内容生成 |
| 知识问答 | 知识点级别的互动聊天 |
| 练习生成 | 生成实践练习和理解测验 |
| 里程碑建议 | 为能力技能生成分级里程碑 |
| 发展策略 | 为专注计划生成发展策略 |
| AI 引导 | 能力画像初始化时的对话式引导 |
| 成长教练 | 小爽助手的跨模块智能对话 |

### 配置方式

在应用内「设置」中管理 LLM 提供者：
1. 添加提供者（选择类型 + 填写配置）
2. 系统自动发现可用模型
3. 各功能自动使用已配置的提供者

## 部署与运维

### 环境要求

- Node.js 18+
- SQLite 3
- PM2 (生产环境)
- Claude CLI (可选，AI 功能)
- Ollama (可选，AI 功能)

### 快速开始

```bash
# 1. 安装依赖
npm install

# 2. 配置环境变量
cp .env.example .env
# 编辑 .env 设置 JWT_SECRET

# 3. 初始化数据库
npm run db:migrate
npm run db:seed-user

# 4. 启动开发服务器
npm run dev
# 访问 http://localhost:3000
```

### 生产部署

```bash
# 一键部署（构建 + 备份 + 迁移 + 重启）
npm run deploy
```

部署流程：
1. `npm run build` — 构建生产包到 `.output/`
2. 自动备份数据库到 `./data/backups/`（保留最近 5 份）
3. `npm run db:migrate` — 应用数据库迁移
4. PM2 重启服务进程

### 数据库管理

```bash
npm run db:generate  # 从 schema 变更生成迁移文件
npm run db:migrate   # 应用迁移
npm run db:studio    # 打开 Drizzle Studio 图形界面
```

- 数据库文件：`./data/assistant.db`（可通过 `DATABASE_PATH` 环境变量配置）
- WAL 模式，支持并发读取
- 外键约束已启用

### 日志与监控

```bash
npm run prod:logs    # 查看 PM2 日志
```

- 日志文件：`./logs/pm2-*.log`
- 请求日志：中间件自动记录所有 API 请求
